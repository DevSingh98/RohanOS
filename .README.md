# RohanOS

**RohanOS** is a custom, bare-metal operating system built from the ground up, named in honor of my beautiful son. The project is a journey into low-level systems programming, starting with the simplicity of the Raspberry Pi Zero and aiming toward modern 64-bit ARM architectures.

## Project Vision
The goal of RohanOS is to evolve from a simple hobbyist kernel into a portable operating system capable of running on:
- **RohanOS/32**: Legacy support for the Raspberry Pi Zero (ARMv6).
- **RohanOS/64**: Modern support for Raspberry Pi 5 (ARMv8-A).
- **RohanOS/Air**: Experimental support for ARM-based laptops.

## Current Progress: Phase 1 (Pi Zero)
- [x] Bootstrapping (Assembly to C++ transition)
- [ ] GPIO Driver (ACT LED Blinking)
- [ ] UART Driver (Serial Console)
- [ ] System Timer & Interrupts

## Getting Started
### Prerequisites
You will need the ARM cross-compiler. On macOS (Apple Silicon), install it via Homebrew:

```bash
brew tap armmbed/formulae
brew install arm-none-eabi-gcc
```

## Building
Clone the repository and run the following command in the root directory:

```bash
make
```

This will generate kernel.img. To boot, copy this file along with the Raspberry Pi firmware (bootcode.bin and start.elf) to a FAT32 formatted microSD card.
